#### Units
All inputs and outputs must define their units with them. Standard set of units is below

Energy:	MeV \
Time:	ns \
Distance:	mm \
Field:	T \
Angles:	rad \
Frequency: MHz \
Density: g/cm3 \
Current: A/mm2 \
Beam format: Px and Py, not x', y' \
RF Phase: 0 degrees bunching, 90 degrees on-crest \
Coordinates: Cartesian
